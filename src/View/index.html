Acerca del sitio

Esta pagina esta diseñada para el uso de distintos administradores (bibliotecarios) que realizan inventario de prestamos de libros, donde se puede ver los libros totales dentro de la biblioteca con detalles tals como: nombre, autor, editorial, año, paginas 7 y estado.

De manera que haciendo uso de un archivo JSON se pueden cambiar el estado de un libro, generando un prestamo o la devolucion de un libro. Dentro de este aplicativo web se hacen uso de herramientas de desarrollo web tales como NodeJS, EJS, Express, JavaScript y Bootstrap.
