Para una mejor ejecución de esta pagina desarrollada se recomienda hacer uso de la linea de comando en la terminal ubicado en la carpeta del proyecot:
'npm run dev'
Linea que apartir del uso de nodemon ejecuta el servidor para la ejecución del taller, alojada en el puerto 3000 como se menciona en la consola al hacer ejecución.
Si se desea hacer uso de un puerto distinto a este se recomienda hacer cambio dentro de la carpeta 'src' en el archivo 'index.js' en la linea 6 donde se encuentra el puerto a usar para la aplicacion
